Functions

First lecture starts at @34:40.

A set comprehension is really just a map and filter. @1:06 one-to-one the definition he uses is the contrapositive of 'if a is not equal to b, then f(a) does not equal f(b)'. The main thing to remember from this lecture is that if two functions are composable, then they have an inverse, and they are only invertible when having the properties one-to-one and onto. This will come up again when we learn invertible matrices. Reading the book The Function chapter, we covered it all in the lecture except an exercise to prove Lemma 0.3.20 that if f and g are invertible, and composed, then the compose is also invertible.

To figure this out let's go back in the book and walk through all the proofs. The first was Proposition 0.3.12 and it's using 0.3.5 Composition of Functions definition to rewrite into h((g of f)(x)), then same definition to rewrite to h(g(f(x))) and finally the same form as proposition 0.3.12. Key here is to see (h of (g of f)(x)) being (h of (function)(x)) since (g of f) is really just a single function.

Next proof is Lemma 0.3.16 a contradiction, suppose f is invertible without being one-to-one, then if we invert f the image will point to two different inputs, which can't be possible if the function is invertible. Unless of course both those inputs are equal, as per the definition if a = b then f(a) = f(b) and in this case we really just have f(a) then since a = b so a = a.

Next proof is Lemma 0.3.17 another contradiction where it's basically telling us in order for something to be invertible it has an image with every element mapped to an unique input.

Next proof is Theorem 0.3.18 using the above two lemmas to define a function g whose domain is the output of f. Then composing these two will result in identity functions as we get the input returned. Finally 0.3.19 strategy is take two inverses then show they must be equal thus only 1 inverse per function if they are invertible.

To prove Lemma 0.3.20 using figure 4 example, if you compose first f(g(1)) you get p which is 1 when inverted. If you compose two inverted functions g(f(x)) then \(f^{-1}(p)\) is \(g^{-1}(a)\) or 1 and both are equal. Read his previous proofs and see if you can mimic them to prove that composing then inverting is equal to two composed inverted functions.
