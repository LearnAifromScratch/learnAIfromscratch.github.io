Cambridge Lecture 1

We have access to the first Cambridge lecture here where he covers a review of calculus we're supposed to know. You can watch this after Calculus 101 if you want it doesn't matter.

First definition: the limit of the ratio f(a + h) - f(a) over h as h gets closer to 0 is the derivative. 'x naught' or x0 notation means the same thing as f(a) they both represent a specific point/input but this is arbitrary (any point) so it's \(f(x_0)\) or \(f(a)\). The h represents a tiny decimal that is tending towards h. Here are the LH and RH limits.

Lets go through the f(x) = |x| example. Plug that into the derivative equation for f(0) = |0| we get (|0 + h| - |0|)/h or |h|/h. However h is negative because it's approaching 'negative 0' from the left side where all the x values are negative so it's (h/-h) or -1.

Try an example for little-o where f(x) = x and g(x) = x2 so f(x)/g(x) is x/x2 = 1/x and if you crank inputs x to infinity then the limit is 0.

Try an example for big-O where f(x) = x and g(x) = x then f(x)/Mg(x) where M is some constant is x/Mx = M or their ratio differs by a constant M meaning g(x) bounds f(x) by a constant factor. He remarks the notation O(g(x)) means an entire class of functions. O(x) represents all linear functions, O(x2) all quadratic functions, O(1) all constant functions. The extension to infinity x > X means some input to f(x) and g(x) past a threshold input. If f(x) = x + 5, m = 1, and g(x) = x2 then for x > 3 f(x) <= Mg(x) and f(x) is in O(x2) though that is a loose upper bound when the constant M can be juked to show x is within linear O(x). Notice the definition allows us to set the constant M to any value we want and is solely included to eliminate constants from big-O notation. If f(x) = 1000x and g(x) = x then f(x) is in O(x) you don't write O(1000x) as any arbitrary constant is included in the definition. This means O(x) + O(x) or O(2x) is still O(x).

Example 3 if f(x) = x2 then plugging it into the derivative equation:

  • \(\frac{(x + h)^2 - x^2}{h}\)
  • \(\frac{x^2 + 2xh + h^2 - x^2}{h}\)
  • \(\frac{2xh + h^2}{h}\)
  • = 2x + h then take the limit of h to zero
  • = 2x + 0

Since h is tending towards 0, before we take the limit we know the remainder term h2 is going to be so small it will approach 0 before we take the limit of h to zero so this is 2xh/h + o(h) though usually h2 is just ignored/junked. Try examples if h = .0001 then check what (.0001)2 is.

Some rules are introduced like chain/product rules. He says to look over the notes and see how big-O is used in the proofs.

Invent derivatives yourself

Go to desmos.com and make a graph f(x) = x2 and notice this is not linear because it does not have the superposition property. A (line)ar line would be a straight line not a curved line meaning if you scaled it with multiplication such as 3(f(x)) and f(3x) the result would be the same. The input f(2) + f(2) and f(2+2) would also be the same if this were linear.

How can we make a linear approximation? Let's try super zooming in on a point in that graph so all the surrounding points in it's neighborhood appear to be sitting on a straight line. This is like when you are at the beach and notice the distant ocean horizon is a straight line even though the earth is curved that's because you're standing on a super zoomed point. Add the point (2, 4) and the point (2.001, 4.004) then zoom that point as I have done here.

When you step off the graph to the horizontal right direction of the x-axis by 0.001 you have to move up towards the y-axis 0.004 in order to meet the graph again. Let's call these dx = 0.001 and dy = 0.004 and to go from dx to dy it is 4dx or f(2 + dx) = 2 + 4dx. If you try this for (3.001, 9.006) then f(3 + dx) = 3 + 6dx and f(4 + dx) = 4 + 8dx.

Taking all the examples, and looking at the notation f(x + dx) = x2 + dx this is really y + dy = y + (2x)dx once function application is done. This means y - y + dy = (2x)dx or dy = (2x)dx or dy/dx = 2x. What is 2x? It was the rise/run or tangent slope and is the derivative of that function.

We have discovered that f(x) = x2 is f'(x) = 2x where that ' tick denotes derivative or using our differentials notation \(\frac{dy}{dx}x^2 = 2x\)

To guarantee you will always have a straight line approximation these dx quantities have to be so small they are infinitesimal or a tiny amount such as dx = 0.001 but instead of 2 zeros you add 'infinitesimally many zeros' meaning dx has the property that it's smaller than the nucleus of an atom but not actually zero. The product of two infinitesimals is zero since if you recall decimal multiplication you would shift the decimal point left adding an infintesimal amount of more zeros to an existing infintesimal and now it's just zero since it is indistinguishable from nothing. Adding two infinitesimals gives another infinitesimal since you aren't moving the decimal point. The problem with this definition is there does not exist any numbers with the property that when you multiply a non-zero quantity with another non-zero quantity it becomes zero. Thus real numbers and limits were invented so now we can pretend these infinitesimals are approaching zero and take their limit as 0.

In single variable aka scalar calculus a linear approximation is a 2D tangent or the straight line magnification. In vector or multi-variable calculus we now have 3D objects and their linear approximation is a magnification all around a (x, y, z) point who's neighbors are now on a flat surface or an entire 2D plane. You use something called a gradient which converts vectors to a scalar and in this case the output tells you where the 2D plane is orthogonal (90 deg) and is the z-axis coordinate of where this plane surrounding the point is as depicted in GeoGebra.

There's formulas for derivatives if you search for them like every power function \(x^n\) is \(nx^{n-1}\) example x2 is 2x1 meaning constants are wiped out such as f(x) = 3 because this is really f(x) = 3x0 and of course when you lower the 0 exponent and multiply the expression by 0 it's gone.

Integration

Honestly derivatives are now all computed with a compiler trick (if the derivative exists, often it doesn't) and the only calculus you will ever do by hand is likely integration for both this course (differential eqs) and probability and statistics. "By hand" means you will still use a program to do it like Mathematica or whatever but will have to know what it all means.

The original purpose of integration was to undo a derivative so if the first derivative of x2 is 2x and the second derivative of 2x is 2 then integrating twice from the second derivative will get you back to the original function x2 and in a physics interpretation this has meaning like the first derivative of time/position is velocity and the second derivative is acceleration (then the jerk, the snap, etc) and you can move between them using integration. They call this the 'fundamental theorem of calculus' in undergrad but you will never see this term used in grad school.

This of course requires you to already know the derivative of a function in order to undo it using an integral. If you don't know the original function then you can try using Riemann sums to approximate the area underneath a curve which is a convoluted method taught in every undergrad and high school that was replaced over 100+ years ago by the Lebesgue integral which itself was generalized/replaced by the Henstock-Kurzweil integral in the 1960s.
