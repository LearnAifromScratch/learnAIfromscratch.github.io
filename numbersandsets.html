Ex. 0.8 hexadecimal is base 16

Ex. 0.14 looking at the definition of division: if 1 divides a then a = q1 which means q is equal to a, or 10/1 is 10. If we manipulate the equation multiplying each side by 1/q or 1/1 then we get a logical answer which is a/q = 1 or a/1 = q so 10/10 = 1 and 10/1 = 10. However if we balance the equation for 0 multiplying each side by 1/q or 1/b we get: 0/q = b or 0/b = q so both q and b are zero which makes no sense since we are supposed to prove every integer can divide 0 however 0/0 is not defined so since q must be 0 then 0/q = b can't exist however 0/b = q can exist.

Proposition 0.15 I proved myself before reading the book proof which you should also do: if b = qc and a = rb then a = r(qc) and this is a = (rq)c or a = (int)c so long as multiplication of two integers is closed in Z which it is. This is a standard proof in Tao's Analysis I that I already read many times.

Ex 0.16 (qd)u + (qd)v = qd is obviously divisible by qd and this is (value) = qd if you've done any programming or (integer) = qd so long as addition is closed in Z.
